ESG Sentiment Analyzer

Project by Swaroop Shivarai Teli

🌿 What It Does

This project analyzes ESG-related news headlines to assess sentiment trends across environmental, social, and governance topics. It uses keyword filters, sentiment scores, and dashboards to make unstructured news data actionable.

🧰 Tools & Techniques

  • Python, Pandas, NLTK/VADER
  • Text Cleaning + Preprocessing
  • Sentiment Scoring
  • Streamlit Dashboard with charts

📈 Outcome

The dashboard provides a real-time view of ESG sentiment in the media, helping stakeholders track trends and reputational risk by theme or keyword.
