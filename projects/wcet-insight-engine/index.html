WCET Insight Engine

Project by Swaroop Shivarai Teli

⚙️ What It Does

This project predicts the Worst-Case Execution Time (WCET) for embedded systems loops using machine learning. It analyzes code metrics to estimate performance boundaries, helping optimize safety-critical software.

🧠 Techniques & Tools

  • Python, Pandas, Scikit-learn
  • Regression models: Decision Tree, Random Forest
  • SHAP Explainability
  • Streamlit App for visual predictions
  • Custom feature engineering from WCET benchmarks

🎯 Outcome

Predicted loop bounds and WCET with high accuracy. Built an interactive interface for real-time code analysis and decision-making insights.
