Supply Chain Delay Predictor

Project by Swaroop Shivarai Teli

What it Does

This project predicts potential supply chain delays using a machine learning model. It uses features like delay probability, shipping info, supplier location, and risk levels to flag shipments at risk.

Key Tools & Methods

  • Python, Pandas, Scikit-learn
  • Classification Models (Random Forest, XGBoost)
  • SHAP for explainability
  • Streamlit Dashboard

Outcome

Built and deployed a real-time delay risk dashboard, helping logistics managers identify risky shipments early and act proactively.
